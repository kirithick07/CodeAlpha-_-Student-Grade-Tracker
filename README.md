# CodeAlpha-_-Student-Grade-Tracker

Student Grade Tracker

Project Description

Student Grade Tracker is a Java console application developed to manage student grades and generate a summary report.

Features

- Input student names
- Input student grades
- Calculate average score
- Find highest score
- Find lowest score
- Display pass/fail status
- Generate grade letters
- Display summary report

Technologies Used

- Java
- Arrays
- Scanner Class
- VS Code
